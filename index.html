SICCA

Sistema de Conciliación de Calificaciones Académicas
¿Qué hace este sistema?
Permite validar y comparar notas finales entre Blackboard y Atenea, identificando inconsistencias, registros faltantes y duplicados.
Pasos de uso:
  • Cargar archivo Blackboard (IBB)
  • Cargar archivo Atenea (IA)
  • Presionar Procesar
  • Revisar resultados
  • Descargar reporte final
Puede ser .xls o .xlsx. Si es .xls, el sistema realiza conversión automática antes de comparar.
Solo se procesan registros con Final Mark = Yes.
